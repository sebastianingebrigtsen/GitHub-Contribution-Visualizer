GitHub Contribution Visualizer

Dark Mode

Slik bruker du applikasjonen

  1. Lag et repository på GitHub:

    Gå til GitHub og opprett et nytt repository.

  2. Generer scriptet:

    Design mønsteret ditt ved å klikke og markere cellene i gridet. Klikk på Generer Script for å få koden.

  3. Opprett en .sh-fil:

    Lag en ny fil på maskinen din, for eksempel contributions.sh. Lim inn det genererte scriptet i denne filen.

  4. Endre repository-URL:

    Finn linjen i scriptet som sier:

    git remote add origin <URL-til-repository>

    Bytt ut <URL-til-repository> med URL-en til ditt GitHub-repository.

  5. Gjør scriptet kjørbart:

    Åpne terminalen, naviger til mappen der scriptet ligger, og kjør:

    chmod +x contributions.sh
  6. Kjør scriptet:

    Kjør scriptet med følgende kommando:

    ./contributions.sh
  7. Sjekk bidragsgrafen på GitHub:

    Gå til profilsiden din på GitHub for å se ditt tilpassede mønster.

Tips:

Sørg for at e-postadressen i Git-konfigurasjonen din matcher den som er knyttet til din GitHub-konto:

git config --global user.email "din.email@eksempel.com"
Kopiert til utklippstavlen!
